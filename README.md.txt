# Dating App - Basic Web Application

This project implements a basic user registration system for a dating application.

## Features
- User registration with username, email, and password
- In-memory storage of user data

## Learnings
- Implemented basic servlet for handling HTTP requests
- Used JSP for creating dynamic web pages
- Practiced dependency injection for better testability
- Implemented unit tests for DAO and servlet classes
- Learned about the importance of input validation in web applications

## Next Steps
- Implement user login functionality
- Add persistent storage (database) for user data
- Enhance input validation and error handling
